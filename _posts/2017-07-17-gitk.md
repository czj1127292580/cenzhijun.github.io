#####打开终端敲入以下命令

```
brew update

brew install git
```
#####如果提示你连接git错误，提示你没有权限，可以执行如下操作：
`which git`
#####如果终端显示如下，则安装成功，执行gitk:
`/usr/local/bin/git`
#####如果安装失败，执行如下操作:
`brew doctor`
